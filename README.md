
# Recipe Manager App
Recipe Manager helps users store and manage their recipes, track cooking history, and generate shopping lists.

## Features
- View recipe statistics.
- Add new recipes.
- Search, filter, and sort recipes
- Display recipes in a neat card format.
- Generate a shopping list based on the selected serving size
- Track cooking history
